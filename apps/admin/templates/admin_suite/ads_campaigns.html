{# ═══ How Campaigns Work ═══ #}

Types: Direct = your own ads, Affiliate = partner products, Network = 3rd party (AdSense etc), House = self-promotion.

Budget: Total spend limit for the campaign. Use $0 for unlimited.

Caps: Daily Cap limits impressions per day, Total Cap limits lifetime impressions. 0 = unlimited.

Priority: Higher priority wins ties when multiple campaigns compete for the same placement.

Weight: Relative probability in rotation. Weight 2 is shown 2x more than weight 1.

Schedule: Only serves ads within the start/end window. Leave blank for always-on.

Live Status: A campaign is LIVE when Active = on, Start ≤ now, and End ≥ now (or not set).

Workflow: Create Campaign → Add Creatives → Assign Creatives to Placements → Campaign goes LIVE.
