{# ═══ How Placements Work ═══ #}

Placement = a named slot in a template where ads appear. E.g. "header-banner", "sidebar-top", "in-content".

Slug: Unique identifier used in templates: {{ "{% render_ad 'header-banner' %}" }}

Context: Page type filter. E.g. "blog" means this slot only appears on blog pages.

Allowed Types: Which creative types can be assigned. Leave empty for any.

Allowed Sizes: Ad dimensions accepted. Leave empty for responsive.

Toggle: Enable/Disable controls whether this placement slot renders ads. Disabled = blank space.

Creatives Count: How many creatives are assigned to this slot via PlacementAssignment.

Workflow: Define Placement → Create Creatives in Campaign → Assign Creatives to this Placement → Rotation engine serves ads.
